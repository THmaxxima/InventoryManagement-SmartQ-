Inventory Management (FC.IVM)
Overview

M2 Inventory Movement Module (Project 4.0) (IVM)

วัตถุประสงค์ :

  • นำเข้า/Unload domistic/Unload import .
  • ย้าย/Move
  • ตัดจ่าย(WP)               
  • Rollback tentative transaction data

 ระบบ IVM พัฒนาให้สามารถรองรับการทำงานในโหมด Mobie เป็นหลัก (แทปเลต)  เพื่อความสะดวกในการปฎิบัติงาน ในหน้างานจริง และยังสามารถ ใช้งานได้บนคอมพิวเตอร์ทั่วไป(PC) ได้พร้อมกัน.ใช้เทคโนโลยี Mapcontrol ในการพัฒนา เพื่อให้สอดคล้องกับ การทำงาน แบบ Mobie และอุปกรณ์ระบบสัมผัส.

 

คำนิยาม/คำศัพท์เฉพาะ :

  • IVM : ชื่อย่อ ของระบบ Inventory Management
  • Site : โรงงาน/โรงผลิต ที่นำระบบ IVM ไปติดตั้งใช้งานอยู่ เช่น วังศาลา, บ้านโป่ง ฯ
  • ลาน : กลุ่มพื้นที่ ที่ใช้ในการจัดเก็บวัตถุดิบ ของแต่ละ Site  เช่น  ลาน1, ลาน2, ลาน3
  • Group area : ชื่อหมวดหมู่ของพื้นที่จัดเก็บ วัตถุดิบ ในแต่ละ ลาน  เช่น  A1
  • Sub area : ชื่อของพื้นที่จัดเก็บวัตถุดิบ (Storage) ซึ่งเป็นพื้นที่ทางกายภาพ ใช้ในการจัดเก็บวัตถุดิบ  เช่น  A1.0, C2.0, D3.0
  • Unload domistic : หลุมโหลดวัตถุดิบในประเทศ  เป็นพื้นที่ สำหรับนำวัตถุดิบลงจากรถ และนำเข้าไปเก็บไว้ ในพื้นที่จัดเก็บ
  • Unload import : หลุมโหลดวัตถุดิบที่นำเข้าจากต่างประเทศ  เป็นพื้นที่ สำหรับนำวัตถุดิบลงจากรถ และนำเข้าไปเก็บไว้ ในพื้นที่จัดเก็บ
  • Other area : ชื่อของพื้นที่จัดเก็บวัตถุดิบ ซึ่งเป็นพื้นที่ทางกายภาพ ใช้ในการจัดเก็บวัตถุดิบที่ไม่สามารถนำเข้าไปเก็บไว้ในพื้นที่จัดเก็บปกติได้(เปรียบเสมือน Sub area)
  • พื้นที่หน้าลาน : ชื่อของพื้นที่จัดเก็บวัตถุดิบ ซึ่งเป็นพื้นที่ทางกายภาพ ใช้ในการจัดเก็บวัตถุดิบที่รอการนำเข้าสู่สายการผลิต
  • Root area : ชื่อของพื้นที่จัดเก็บวัตถุดิบ ซึ่งเป็นพื้นที่ทางกายภาพ ใช้ในการจัดเก็บวัตถุดิบที่แตกเสียหาย
  • WP : กระบวนการนำวัตถุดิบเข้าสู่สายการผลิต
  • Souce area : พื้นที่จัดเก็บวัตถุดิบต้นทาง(Sub area) ใน Process ต่างๆ
  • Destination area : พื้นที่จัดเก็บวัตถุดิบปลายทาง(Sub area) ใน Process ต่างๆ
  • Max capacity : จำนวนน้ำหนักของวัตถุดิบ สูงสุดที่สามารถจัดเก็บได้ในแต่ละ Root area
  • Inuse capacity : จำนวนน้ำหนักของวัตถุดิบที่จัดเก็บ คิดเป็นเปอร์เซ็นต์ต่อ Max capacity
  • Weight ticket : เลขที่ตั๋วชั่ง เป็นข้อมูลของยานพาหนะ และวัตถุดิบ ที่รอนำเข้าไปสู่กระบวนการจัดการวัตถุดิบต่อไป ใช้ใน Process unload
  • Zoom level : เป็นระดับ การแสดงผลข้อมูลในหน้า frm_IVM_Main_Layout มีอยู่ 3 เลเวล
  • Zoom level 1 : เป็นเลเวล มาตรฐาน ในเลเวล1 จะแสดงผลในส่วนของ รายละเอียด, สีของสถานะ, เวลาล่าสุดของการอัพเดทข้อมูลจากฐานข้อมูล, และตำแหน่ง ของ Group area, Unload ในแต่ละ ลาน และสามารถใช้งาน Process ต่างๆ ได้ในเลเวลนี   
  • Zoom level 2 ในเลเวลที่ 2 จะแสดงรายละเอียดของ วัตถุดิบที่จัดเก็บอยู่ และสามารถกำหนดสะถานะ Capacity ได้ในเลเวลนี้
  • Zoom level 3 : ในเลเวลที่ 3 จะแสดงรายละเอียดเหมือนกับ เลเวล2 แต่จะมีไอคอน เพิ่มเติมขึ้นมา เพื่อให้สามารถ กดเพื่อดูรายละเอียดของ ชนิดวัตถุดับ และรายละเอียดที่เกี่ยวข้อง ของวัตถุดิบที่ทำการจัดเก็บ
  • หน้าจอหลัก : เป็นชื่อที่ใช้เรียกแทน frm_IVM_Main_Layout
